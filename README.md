# Trividhi Labs

> **Your Trusted Partner for secure 'Digital Identity' & 'Verifiable Credential' solutions**

The official website of **Trividhi Labs Private Limited** — a digital identity and verifiable credentials platform built for healthcare and education ecosystems.

## 🌐 Overview

Trividhi Labs delivers tamper-proof, privacy-centric digital identity infrastructure powered by globally accepted standards. The platform enables real-time verification, fraud prevention, and seamless portability of credentials across institutions and geographies.

---

## ✨ Core Capabilities

- 🔐 **Trusted Data Verification** — Cryptographically verifiable records
- 🛡️ **Tamper-Proof by Design** — Immutable credential issuance
- 🚫 **Built-in Fraud Prevention** — End-to-end integrity checks
- 🔒 **Privacy-Centric Architecture** — User-controlled data sharing
- 🔄 **Seamless Portability** — Credentials that travel with the user
- 🌍 **Built on Global Standards** — W3C VC, DID, and open protocols

## 🚀 Getting Started

### Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/research-prj26/trividhi-labs.git
   cd trividhi-labs
   ```

2. Open `index.html` directly in a browser, **or** use a local server:
   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js (npx)
   npx serve
   ```

3. Visit `http://localhost:8000` in your browser.
